# Code Formatter

A simple tool to format your code beautifully.

## Usage

```bash
python scripts/format.py <file>
```

## Supported Languages

- Python (.py)
- JavaScript (.js)
- TypeScript (.ts)
- JSON (.json)

## Examples

Format a Python file:

```bash
python scripts/format.py main.py
```
